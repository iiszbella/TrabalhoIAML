Tema do ChatBot:  
Nightware é um chatbot especialista em jogos digitais de terror, como Resident Evil, Outlast e outros títulos do gênero.

Tecnologias utilizadas:
Python para implementação
Flask como framework web
LLM Qwen como modelo de linguagem

Descrição:  
O projeto consiste na construção de um chatbot capaz de interagir com usuários sobre jogos digitais de terror, oferecendo informações, curiosidades e recomendações. A arquitetura combina técnicas de IA e ML com o uso de um modelo de linguagem de larga escala (LLM), garantindo respostas mais naturais e contextualizadas.
